Portfolio Template for Mohammed Sinwan

This template includes a Streamlit app (streamlit_app.py) generated from the uploaded resume PDF located at:
/mnt/data/sinwan-B14.pdf.pdf

How to run locally:
1. Create a virtual environment (recommended)
2. pip install -r requirements.txt
3. streamlit run streamlit_app.py

Files:
- streamlit_app.py : Main Streamlit portfolio app
- requirements.txt : Python requirements

Notes:
- The app will attempt to offer a download button for the original PDF resume (it references the path used to generate the template).
- Tweak the fields in streamlit_app.py to customize styling, images, and content.
